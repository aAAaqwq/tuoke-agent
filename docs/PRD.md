## 1. 产品定义

### 一句话描述
**AI 销售员** — 用户定义目标客户 → Agent 自主完成线索发现→画像→触达→维护→成交辅助全闭环。

### 核心本质
拓客智能体 ≠ 找线索工具。它是一个完整的 AI 销售 Agent：
1. 找到潜在客户
2. 画出客户画像（行业/规模/痛点/需求）
3. 通过合规渠道触达（邮件→企微→更多）
4. 持续维护客户关系
5. **最终目标：提高成交率**

### 目标用户
**中国中小企业老板** — 有产品/服务，但缺客户或目标客户画像不清晰。

### 与竞品的本质差异

| 维度 | 探迹/小蓝本 | Clay/Apollo | **我们** |
|------|-----------|-------------|---------|
| AI 能力 | 规则引擎 | AI 辅助 | **AI Agent 自主决策（Level 3）** |
| 定价 | 3-20万/年 | $149-349/月 | **$29-99/月** |
| 中文支持 | ✅ | ❌ | **✅ 原生** |
| 企微集成 | ❌ | ❌ | **✅** |
| 核心壁垒 | 数据量 | 多源集成 | **AI Agent + 中文 + 价格** |

## 4. 五大难点及突破方案

> 详细方案见 `docs/challenge-solutions.md`

### 4.1 数据源获取 🟡→🔴
**问题**: 企业联系方式被企查查/天眼查垄断，无免费高质量源
**方案**: 分层策略 — 付费API(主) + 公开采集(辅) + 用户导入(补充) + 社区贡献(长期)
**关键决策**: 是否购买企查查 API（约 3000-10000 元/年）

### 4.2 私域引流 🔴
**问题**: 小红书/脉脉反爬极严，批量操作封号风险大
**方案**: Phase 1 先走邮件+企微（合规可控），平台引流需验证后再扩展
**Phase 1 不做**: 私域平台自动化

### 4.3 冷启动 🔴
**问题**: 无历史成单数据，评分模型参数无法校准
**方案**: 行业公开数据初始化 → 人工校准 → 半监督学习 → ML 三阶段渐进
**MVP 策略**: 先用规则引擎 + 行业知识库，不依赖 ML

### 4.4 合规风险 🔴
**问题**: 个人信息保护法 + 数据采集合法性
**方案**: Privacy-by-Design + PII 检测脱敏 + 数据生命周期管理 + 小law 审查
**铁律**: 只用公开数据+授权数据，不做任何非法采集

### 4.5 LLM 成本 🟡
**问题**: Agent 编排多层调用，Token 消耗大
**方案**: 四层优化
1. 分层模型（DeepSeek 日常 / GPT-4o 关键决策 / 本地模型辅助）
2. 语义缓存（相似查询命中缓存，省 50%+ Token）
3. Prompt 压缩（精简指令，去掉冗余描述）
4. 预算熔断（单用户月消耗上限）

## 5. 验收标准

### Phase 0 验收
- [ ] 确定至少 1 个可用数据源（付费或免费）
- [ ] 爬取 100 条企业数据样本
- [ ] 合规审查报告（小law 出具）
- [ ] 反爬机制调研报告

### Phase 1 MVP 验收
- [ ] 输入行业+地区 → 采集 100+ 线索
- [ ] 自动清洗去重 → 有效率 ≥80%
- [ ] AI 画像生成 → 包含行业/规模/痛点/需求 4 维度
- [ ] BANT 评分 → 输出优先级排序
- [ ] 输出：**企业画像报告**（交付给用户）

### Phase 2 验收
- [ ] 邮件发送 50+ 封 → 送达率 ≥90%
- [ ] 开信率 ≥25%
- [ ] 个性化内容 → 每封含 2+ 个性化元素
- [ ] Dashboard 显示核心指标

### 性能基线
- API P95 < 200ms
- 前端 LCP < 2s
- 可用性 99.5% SLA（MVP 阶段）

| Phase 0-1 MVP | ~$100 | LLM API + 服务器 |
| Phase 2 | ~$200 | + 邮件服务 + 数据API |

### 定价

| 套餐 | 月费 | 线索/月 | 核心功能 |
|------|------|---------|---------|
| **Starter** | $29 | 300 | 线索发现 + 画像报告 |
| **Pro** | $49 | 1,000 | + 邮件触达 + 效果追踪 |
| **Business** | $99 | 无限 | + 企微 + 多租户 + API |

**Break-even**: 3-7 个付费用户

### ROI 计算
```
传统拓客平台: 3-20万/年 + 人工销售成本
拓客智能体:   $29-99/月 = ¥200-700/月
节省: 90%+ 成本

AI Agent 价值:
- 24/7 自动线索发现（替代 1 个 SDR 的 30% 工作量）
- 自动画像 + 评分（替代分析师 50% 工作量）
- 个性化内容生成（替代文案 40% 工作量）
```
